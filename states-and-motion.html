Reduced motionIf prefers-reduced-motion = reduce, freeze on F.01, drop the pulse, keep the layout. No alternative animation, no fade.
Where usedLanding Ch.02 · 404 routing block · Mobile Ch.02 · Field Note inline figure. All four instances run the same five-frame loop in lockstep — they share a single timing source.
ImplementationPure CSS keyframes per row + a 4.0s `@keyframes routerWalk` master. No JS unless an IntersectionObserver is needed to pause when off-screen (recommended on Field Note article).
