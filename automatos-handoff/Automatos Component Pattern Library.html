Spacing Scale
Vertical spacing follows an 8px grid system. Use space-y-6 between page sections, space-y-4 within sections.
Vertical Rhythm
Consistent spacing between page elements.
Context Class Value Use Case
Page sections space-y-6 24px PageHeader → StatsBar → Tabs → Content
Within section space-y-4 16px Cards in grid, list items, related content
Dense content space-y-3 12px Form fields, compact lists
Generous spacing space-y-8 32px Between unrelated major sections

Typography Scale
Consistent type sizing across the platform. Page titles are 28-32px, body text is 14px, labels are 12px.
Type Scale
Text sizing hierarchy.
Element Class Size Weight Use Case
Page Title text-2xl md:text-3xl 24px → 30px 700 (bold) PageHeader titles
Section Title text-xl 20px 600 (semibold) Card headers, modal titles
Body Text text-sm md:text-base 14px → 16px 400 (normal) Paragraphs, descriptions
Small Text text-xs 12px 400 (normal) Labels, helper text, captions
Stat Value text-2xl 24px 700 (bold) Stat card values
Icon Sizing Scale
Icons follow a 4px scale: 16px for inline text, 20px for buttons, 24px for headers, 32px for empty states.
Icon Scale
Consistent icon sizing relative to surrounding text.
Context Class Size Example
Inline text w-4 h-4 16px Paragraph icons, table cells
Buttons, tabs w-5 h-5 20px Button icons, tab icons, nav items
Headers, titles w-6 h-6 24px Page header icons, section titles
Empty states w-8 h-8 32px EmptyState component, feature callouts
Large illustrations w-16 h-16 64px Welcome screens, onboarding
